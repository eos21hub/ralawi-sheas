# Ralawi Sheas — Marketing Site

A premium marketing website for **Ralawi Sheas**, a 100% pure unrefined shea butter brand hand-pressed in northern Ghana. React + Vite + TypeScript + Tailwind + Framer Motion + Lenis.

Aesthetic target is restrained, tactile luxury — Aesop / Le Labo / Glossier rather than SaaS-template glossy. Light theme overall, with strategic dark sections to lift visual rhythm.

## Commands

| Command | What it does |
|---------|--------------|
| `npm run dev` | Vite dev server with HMR |
| `npm run build` | Type-check (`tsc -b`) + production build to `dist/` |
| `npm run preview` | Serve the built `dist/` for verification |
| `npm run optimize:images` | Convert `src/assets/source/*.JPG` → `src/assets/products/*.webp` (run after dropping new source images) |
| `npm run compress:video <src> <name>` | Compress a raw video into web-ready WebM + MP4 + poster. Example: `npm run compress:video raw/hero.mp4 hero` → writes `public/media/hero.{webm,mp4,jpg}` |

## Design tokens

Defined in `tailwind.config.ts`:

- **Colors:** `cream` (`#FBF8F3`), `cream-deep`, `sand`, `espresso` (`#2A1F17`), `bark`, `shea` (`#C89B5A`), `shea-dark`, `shea-light`, `sage`, `sage-deep`. No pure `#000` or `#FFF`.
- **Type:** `font-serif` = Fraunces (display + headlines), `font-sans` = Inter (body).
- **Scale:** Fluid `clamp()` from `text-xs` → `text-display` (hero only, peaks at 6rem). 1.25 modular ratio.
- **Spacing:** Default 8px grid + `15` (3.75rem), `18`, `22`, `30` (7.5rem section padding), `38` (9.5rem ultrawide section padding).
- **Easing:** `transition-out` = `cubic-bezier(0.16, 1, 0.3, 1)`. Framer durations in `src/lib/motion.ts`.
- **Corner radius convention:** Feature images `rounded-3xl`, grid cards `rounded-2xl`, badges `rounded-full`.

## Video pipeline

The Hero section uses a Higgsfield-generated loop (Wan 2.7, 720p, 5s seamless). Build does **not** bundle videos — they're statically served from `public/media/`.

To regenerate or replace:

```bash
# 1. Generate via Higgsfield CLI
higgsfield generate create wan2_7 \
  --prompt "..." \
  --aspect_ratio 16:9 --resolution 720p --duration 5 --wait

# 2. Download the returned URL to raw/hero.mp4

# 3. Compress to web-ready WebM + MP4 + JPG poster
npm run compress:video raw/hero.mp4 hero

# 4. Refresh dev server — no rebuild needed
```

If a video file is missing, the `<video>` element falls back to its `poster` attribute (the matching WebP in `src/assets/products/`).

**Note:** The FinalCTA section does **not** use `<video>` — it uses an animated `<img>` with Ken Burns + drift via Framer Motion. This was chosen deliberately because the all-source-404 fallback behavior of `<video>` is inconsistent across browsers, causing blank sections. Don't swap it back to `<video>` unless you actually have a cinemagraph file 

## Accessibility & motion

- All Framer scroll transforms gate on `useReducedMotion()` and zero out under reduced motion.
- Lenis smooth-scroll is bailed entirely under `prefers-reduced-motion` (browser-native scroll takes over).
- Pinned product showcase collapses to a stacked grid on `<lg` or reduced motion (all 4 products visible).
- Testimonials marquee uses `motion-reduce:animate-none motion-reduce:overflow-x-auto motion-reduce:snap-x` for keyboard / reduced-motion users.
- Headline reveal animations use word-level clip masks with `aria-label` on the parent so screen readers receive the full sentence at once.
- Color contrast: body text (`#6B5A4C` on `#FBF8F3`) ≥ 5.4:1; espresso headlines ≥ 12:1.
- All decorative gradient orbs are `aria-hidden` and `pointer-events-none`.
