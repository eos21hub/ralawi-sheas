# Ralawi Sheas — Marketing Site

A premium marketing website for **Ralawi Sheas**, a 100% pure unrefined shea butter brand hand-pressed in northern Ghana. React + Vite + TypeScript + Tailwind + Framer Motion + Lenis.

The aesthetic target is restrained, tactile luxury — Aesop / Le Labo / Glossier rather than SaaS-template glossy.

## Quick start

```bash
npm install
npm run dev      # http://localhost:5173
```

## Commands

| Command | What it does |
|---------|--------------|
| `npm run dev` | Vite dev server with HMR |
| `npm run build` | Type-check (`tsc -b`) + production build to `dist/` |
| `npm run preview` | Serve the built `dist/` for verification |
| `npm run optimize:images` | Convert `src/assets/source/*.JPG` → `src/assets/products/*.webp` (run after dropping new source images) |
| `npm run compress:video <src> <name>` | Compress a raw video into web-ready WebM + MP4 + poster. Example: `npm run compress:video raw/hero.mp4 hero` → writes `public/media/hero.{webm,mp4,jpg}` |

## Design tokens

Defined in `tailwind.config.ts`:

- **Colors:** `cream`, `cream-deep`, `sand`, `espresso`, `bark`, `shea`, `shea-dark`, `shea-light`, `sage`, `sage-deep`. No pure `#000` or `#FFF`.
- **Type:** `font-serif` = Fraunces (display + headlines), `font-sans` = Inter (body).
- **Scale:** Fluid `clamp()` from `text-xs` → `text-display` (hero only, peaks at 6rem).
- **Spacing:** Default 8px grid + `15` (3.75rem), `18`, `22`, `30` (3.75rem section padding), `38` (9.5rem ultrawide section padding).
- **Easing:** `transition-out` = `cubic-bezier(0.16, 1, 0.3, 1)`. Framer durations in `src/lib/motion.ts`.

## Adding Higgsfield-generated media

1. Generate raw video via `higgsfield generate create seedance_2_0 --prompt "..." --duration 8 --resolution 1080p --wait` (or via the `higgsfield-generate` skill).
2. Download the result locally as `raw/hero.mp4`.
3. Compress: `npm run compress:video raw/hero.mp4 hero` (produces `public/media/hero.webm`, `hero.mp4`, `hero-poster.jpg`).
4. The `<Hero>` component already references `/media/hero.{webm,mp4}` — refresh the dev server and the loop will play.
5. Repeat for the cinemagraph (`name = cinemagraph`).

If the videos aren't present, the components fall back to the WebP poster — the site remains visually intact.

## Deployment

Static-only output (`dist/`). Drop on:

- **Netlify Drop** — fastest: `npm run build` then drag `dist/` to https://app.netlify.com/drop
- **Vercel** — `npx vercel` from repo root, accept defaults
- **Cloudflare Pages** — connect repo, build command `npm run build`, output `dist`

## Accessibility & motion notes

- All scroll-driven motion respects `prefers-reduced-motion` (Lenis is disabled, parallax transforms zero out, marquee freezes, pinned product showcase collapses to a stacked grid).
- Headline animations use word-level clip masks with `aria-label` on the parent so screen readers receive the full sentence at once.
- Color contrast: body text (`#6B5A4C` on `#FBF8F3`) ≥ 5.4:1; espresso headlines ≥ 12:1.
